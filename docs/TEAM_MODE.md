# Team Mode

## What team mode adds

- Workspace setup
- Admin and standard users
- Login flow
- 2FA preparation
- Invitations
- Roles and permissions
- Audit log visibility

## Roles

- `Admin`
  user and team management
- `Editor`
  standard project editing
- `Uploader`
  file-heavy work without full admin rights
- `Reader`
  read-only access

## Admin actions

- Add or remove users
- Change roles
- Toggle upload rights
- Toggle read-only
- Reset 2FA
- Set temporary passwords
- Revoke invitations
